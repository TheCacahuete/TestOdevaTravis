Howto:
------

clonage du paquet:
$~ https://gitlab-etu.fil.univ-lille1.fr/lefebvrer/COO-TP2.git

une fois le paquet clonais, a la racine du paquet faites:
$~ mvn package

vous pouvez lancer l'excecutable ainsi:
$~ java -jar target/TP-Donjon-1.0-SNAPSHOT.jar
